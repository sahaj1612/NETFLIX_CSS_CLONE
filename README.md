# Netflix CSS Clone

Netflix CSS Clone is a static frontend project that recreates parts of the Netflix landing experience using HTML and CSS.

## Tech Stack

- HTML
- CSS

## Highlights

- Netflix-style landing page
- Sign-in style entry section
- Trending content cards
- Multi-page static flow

## Run Locally

Open `netflix.html` in any browser.

## Note

This repository is intended for frontend styling practice and layout recreation.
